# meettafel_software_1

De software die is ontwikkeld in die map verbindt sensoren op de meettafel draadloos met een wifi-accesspoint, via welke real-time de meetwaarden in een browser weergegeven kunnen worden en met een druk op de knop naar csv file gedownload. De sensoren houden deepsleep om stroom te besparen als ze niet gebruikt worden.

Elke sensor kan geprogrammeerd worden met de code in sensor-node (niet vergeten eerst het sensor-id in de code in te vullen).

De collector (het accesspoint) kan gebouwd worden met de software in collector_node. Vergeet niet eerst de files in de bijbehorende data folder met de LittleFS upload plugin te uploaden.
